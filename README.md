# MiniTorch

MiniTorch is a lightweight library that reimplements core deep learning components similar to PyTorch with a unique approach. It provides a tensor structure, an automatic differentiation system, and basic neural network modules within a minimal and understandable architecture. It also has a nearly full stack-based version, where all data is stored in pre-allocated arrays.

*The project is still in development, immature and not optimized.*

## Features

- Original Tensor implementation
- Automatic differentiation (autograd) mechanism
- Basic neural network layers
- Modular and simple structure
